Lesson 5: Making Things Move

MISSION: ANIMATE THE CIRCLE

Welcome, game developer! Your mission is to learn how to create animation in p5.js.

By the end of this mission, you'll be able to make objects move across the screen on their own.

TRAINING MODULE

Watch the video segment from Transcript 2.2 (4:10-7:25)

CODING CHALLENGE

Your mission: Make the circle move from left to right.

let circleX = 100;

function setup() {
  createCanvas(400, 400);
}

function draw() {
  background(220);
  ellipse(circleX, 200, 50);
  circleX = circleX + 1;
}

KNOWLEDGE CHECK

Look at this code:

let y = 100;

function setup() {
  createCanvas(400, 400);
}

function draw() {
  background(220);
  ellipse(200, y, 50);
  y = y + 2;
}

The circle will move downward (increasing y moves down in p5.js) at a speed of 2 pixels per frame.
